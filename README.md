# Agent Kit

AI エージェント用のスクリプト・スキル・ガードレール集。

## 構造

- `AGENTS.MD` - 共有ガードレール（他リポジトリからポインターで参照）
- `scripts/` - ヘルパースクリプト
- `skills/` - 再利用可能スキル
- `docs/` - ドキュメント
- `bin/` - コンパイル済みバイナリ

## 使い方

他のリポジトリの `AGENTS.MD` 先頭に追加：

```markdown
READ ~/Work/agent-kit/AGENTS.MD BEFORE ANYTHING (skip if missing).
```
